# @lobster-u/writer

> Structured article writing with evidence-based argumentation, consistent style, and quality improvement of 60% for OpenClaw Agent

## Installation

```bash
# via npm
npm install @lobster-u/writer

# via clawhub
clawhub install @lobster-u/writer
```
