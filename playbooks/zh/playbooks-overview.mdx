# Playbooks：从内容转向产出

2019：你在囤课程、囤笔记、囤链接。
2026：你用 Agent 交付产出。

Lobster University 是一个面向 **Human‑Agent Loop** 的学习社区。我们用 Agentic Skills 取代传统 EdTech SaaS，让学习直接变成可交付的结果。
我们还在早期，需要社区伙伴一起共创、测试与验证。

Playbook 是一个 20 分钟学习 session，目标是产出真实结果，而不是更多笔记。

## 10 分钟你能做什么
- 选择一个 playbook
- 运行一次 3 步流程
- 保存输出，并判断是否继续

## Playbook 的构建方式（10/90 逻辑）
1. **市场验证的需求**：研究头部 EdTech（如 Duolingo、Coursera），提炼其背后的核心学习原理。
2. **学习科学骨架**：基于权威学习理论（如 ZPD、认知负荷、自我调节学习）。
3. **人机分工**：明确哪些是人类决策、哪些由 Agent 执行，以最大化效率。
4. **技能整合**：整合高信号学习技能，确保可一键下载并落地执行。

## 当前 Playbooks（10 个主题）

## **Skill Pack**（一键下载）
每个 playbook 都包含可一键下载并安装到 Agent 的 **Skill Pack**。
Skill Pack 支撑 playbook 的 Agent 侧执行，让 session 可运行、可落地。

当前覆盖：**62% verified**，共 **104 skills**（64 verified，40 candidate）。

## 贡献（从小处开始）
我们正在与社区一起构建。你可以从三种方式开始贡献：
- 测试 playbook 并分享结果（哪些有效、哪些无效、具体产出）
- 发现学习相关 skills 并提交到库中
- 创建新的 skills 并回馈社区
